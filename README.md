# resume-builder
Web application that will auto-generate a nice and properly formatted Resume from the information filled up in a form.

<h2> 
  Tech used<br> </h2>
  <ul>
    <li>React</li>
       <li>Node.Js</li>
       <li>Express.js</li>
   <li>HTML</li>
       <li>Tailwind CSS ,Bootstrap ,material ui</li>
      
  </ul>
